🛠️ Tech Stack
- Flutter
- Dart
- Material Design Widgets

⚙️ How It Works
- The app uses local dummy data (List) instead of a backend.
- Navigation between screens is handled using Flutter’s Navigator.
- UI components are built with reusable widgets.

🔧 Getting Started
- Clone the repository:
  git clone https://github.com/isminazzsonmezz/Chat_App_with_Flutter.git
- Navigate to project folder:
 cd flutter-chat-app
- Install dependencies:
 flutter pub get
- Run the app:
 flutter run
